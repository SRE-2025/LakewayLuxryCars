# Deep Audit — 4 Reference Sites → Lakeway 20-Page Plan

## 1. Collection Suites (collection-suites.com) — Miami / Palm Beach
- **Nav / pages:** Locations, Features, About, Contact, One of One, Blog, Members (each a separate page).
- **Home structure:** full-screen **video** hero → Available Suites → Suite Features ("Essential Craftsmanship & Vision") → About ("How Racing Through 1950s Europe Created a New Standard") → Members Only ("Sign In. Start Up.") → Let's Connect (contact form) → footer with partners.
- **Sales model:** suites for sale, **pricing from $2M+**, varies by size/location.
- **Form fields:** interested location, type of inquiry (Sales / Commercial), message.
- **Partners strip:** Gumball 3000, Moda Miami, Miami Concours, RM Sotheby's — social proof.
- **Steal:** video hero, "One of One" bespoke page, Blog for SEO, partners/credibility strip, the elegant restraint.

## 2. Concours d'Elegance Texas (concoursdelegancetexas.com) — Lake Travis / Austin
- **Nav / pages:** Home, About, Site Plan, Elevations & Floor Plans, Design Gallery, Contact.
- **Product:** 105 private suites **1,500–6,000 sf** + two **7,000 sf penthouses**.
- **Named models:** The Montecito 6,000 · The Prime 4,500 · The Diplomat 3,000 · The Luxe 2,250 · X-Suite 1,500 — each shown with floor plan + elevation render.
- **Site plan:** building map with **R = reserved, S = sold** icons; "Priority Choice Reservation Phase," 15 units at pre-sale pricing, Buildings 1–8 released.
- **Delivery spec:** 150–250 AMP power, generator, climate HVAC, finished drywall, water, custom upgrades.
- **Amenities:** helipad, private lake access + marina, 5,000 sf owners club, rooftop decks, smart security, gated, 25 min from downtown.
- **Sales motion:** site visit → download reservation packet → submit docs with lender pre-approval; **Texas private lender financing**.
- **Steal:** the whole car-condo sales architecture — named models, R/S site plan, reservation phase, financing, live-work-collect framing.

## 3. T11 Car Condos (t11carcondos.com) — Circuit of the Americas, Austin
- **Nav / pages:** Home, Community, Condos, Concierge, COTA, Partners, Privacy, Terms + **a dedicated page per condo model** (medium / large / large+ / xlarge / xxlarge) + Contact form page.
- **Condo tiers with live counts:** Medium (4 now / 31 phase 2), Large (6 / 2), Large+ (1 / 13), XLarge (5 / 0), XXLarge (0 / 10) — each with a **"Learn more" → its own page**.
- **Availability:** filterable map (All / XXL / XL / L+ / L / M), each filter linking to that model page.
- **Owner benefits:** guaranteed track time (2 wknds/yr), direct F1 track access, private Turn-11 viewing deck, 20% off COTA experiences, future project discounts, exclusive owner events.
- **Amenities:** owner's lounge, resort pool, catering kitchen, conference room, pickleball, EV charging.
- **Steal:** **per-model detail pages** (the key to depth), availability counts, Concierge page, benefits + amenities as rich carousels, clean Webflow polish.
